> - P3 视频合成前**再次执行 L1+L2 复检**
> - P4 交付前**执行 L3 抽检**

## 强制要求（任一不通过则拒绝发布）

### #0 素材唯一性 & 真实性 & 感知去重（最高优先级 🔴🔴🔴）

- **要求**：
  - ✅ **每张素材必须唯一**（N 个场景 = N 张不同图片，严禁重复）
  - ✅ 必须**从网上下载真实截图/照片**（GitHub、官网、新闻、Unsplash/Pexels）
  - ✅ Canvas 生成的 card 图占比 ≤ 20%，且必须标注
  - ✅ **必须通过三级去重防线（L1 MD5 + L2 感知指纹 + L3 视觉抽检）**
  - ✅ **文件扩展名必须与实际格式匹配**（JPEG 不能命名为 .png）
- **禁止**：
  - ❌ 任何图片使用 2 次及以上（即使不同 zoom 效果也不行）
  - ❌ 纯色背景 + 文字卡片作为主要素材
  - ❌ **同图不同压缩/不同尺寸冒充不同素材**（L2 感知指纹必抓）
  - ❌ **一张图复制8次改名为不同场景素材**（v7.1 事故模式）

#### #0-A 三级去重防线（v7.1 强制）

| 级别 | 方法 | 能检测 | 不能检测 | 执行时机 |
|------|------|--------|---------|----------|
| L1 | 完整文件 MD5 | 二进制完全相同 | 同图不同压缩 | P2完成后, P3合成前 |
| L2 | 感知指纹（图像数据区采样MD5） | 同图不同压缩/格式 | 内容相似但不同图 | P2完成后, P3合成前 |
| L3 | 视觉抽检（AI/人工目视） | 语义不匹配、内容相似 | 需要理解含义 | P4交付前 |

#### #0-B L2 感知指纹算法（Node.js 实现）

```javascript
// dedup_v3.js 核心逻辑
const crypto = require('crypto');
function perceptualFingerprint(buffer) {
  let sample;
  if (buffer[0] === 0x89) { // PNG: skip 8-byte header + IHDR(13B)
    sample = buffer.slice(25, 2073); // 跳过元数据，取2KB图像数据
  } else if (buffer[0] === 0xff) { // JPEG: skip header to image data area
    sample = buffer.slice(400, 2400); // 跳过EXIF等，取2KB图像数据
  } else {
    sample = buffer.slice(0, 2048);
  }
  return crypto.createHash('md5').update(sample).digest('hex').slice(0, 12);
}
```

#### #0-C 去重执行流程（强制嵌入 P2/P3）

```
P2 素材生成完成
  ↓
运行 dedup_v3.js（L1 + L2）
  ├─ 发现重复 → 删除/替换重复素材 → 重新检查
  └─ 全部通过 → 写 phase_done_P2.txt（含去重报告）
      ↓
P3 视频合成开始前
  ↓
再次运行 dedup_v3.js（复检）
  ├─ 发现重复 → 回退到 P2 修复
  └─ 全部通过 → 开始合成
```

### #1 素材语义匹配
- **要求**：每张素材必须与对应场景的文案内容**语义相关**

### #2 视频规格（强制横屏）
- **分辨率**：**1920×1080（16:9 横屏）**
- **格式**：MP4（H.264 + AAC）
- **帧率**：30fps
- **禁止**：❌ 竖屏 1080×1920、❌ 方形 1080×1080

### #3 文件大小
- **最低要求**：≥ 1MB
- **推荐**：≥ 5MB

### #4 视频时长
- **要求**：**30-70秒**（30秒以上，70秒以下）
- **禁止**：< 30秒 或 > 70秒

### #10 单个素材显示时长（v7.2 新增 🔴）
- **要求**：**每个图片素材在视频中的显示时长不超过 5 秒**
- **适用范围**：仅限图片场景（视频片段不受此限制）
- **实现方式**：
  - P1 文案阶段确保场景数足够（建议 ≥12 个场景）
  - P3 构建阶段每段图片视频 = min(TTS_该段时长, 5s)
  - 如某段 TTS > 5s 且只有图片素材 → 拆分为 2 个场景
- **验收**：检查 concat 列表中每个图片片段时长，确认全部 ≤ 5s
- **禁止**：❌ 单张图片播放超过 5 秒

### #5 动态效果（8种强制覆盖）
- **要求**：**必须严格使用8种不同 Ken Burns zoompan 效果，按场景index轮流分配**
- **禁止**：所有场景使用同一种效果，或效果种类<8种

| # | 效果 | 表达式 | 分配场景 |
|---|------|--------|---------|
| 1 | zoom-in | `z='min(zoom+0.002,1.3)'` | index % 8 == 0 |
| 2 | zoom-out | `z='max(zoom-0.002,1.0)'` | index % 8 == 1 |
| 3 | pan-left | `x='iw/2-(iw/zoom/2)+0.5*on'` | index % 8 == 2 |
| 4 | pan-right | `x='iw/2-(iw/zoom/2)-0.5*on'` | index % 8 == 3 |
| 5 | pan-up | `y='ih/2-(ih/zoom/2)-0.5*on'` | index % 8 == 4 |
| 6 | pan-down | `y='ih/2-(ih/zoom/2)+0.5*on'` | index % 8 == 5 |
| 7 | pulse | `z='1+0.06*sin(ON)'` | index % 8 == 6 |
| 8 | diagonal | `x-0.3*on, y-0.2*on'` | index % 8 == 7 |

### #6 字幕（v5.3 更新：单行强制 + 音画同步 + 字体大小）

**⚠️ 字幕三大铁律（违反任一即返工）：**

#### 铁律 A：单行显示（禁止多行）
- **要求**：每段字幕**只能显示一行**，禁止换行/多行
- **实现**：SRT中每段只有一行文字，不含`\n`
- **验收**：抽帧检查，确认画面只有一行字幕

#### 铁律 B：音画同步（时间轴严格匹配TTS）
- **要求**：字幕出现时间 = 对应TTS音频开始时间，结束时间 = TTS结束时间
- **实现**：根据TTS每段实际时长计算累积时间轴
- **验收**：播放视频，确认字幕与配音完全同步

#### 铁律 C：字体大小（横屏36-42px）
- **要求**：横屏1920×1080下，FontSize **36-42px**
- **实现**：SRT force_style 或 ASS Style 中设置FontSize=38
- **验收**：抽帧检查，字幕占画面高度约5-8%

**SRT force_style 模板（v5.3）**：
```
Fontname=Microsoft YaHei Bold
FontSize=38
FontColor=&HFFFFFF
BorderStyle=1
OutlineColour=&H000000
Outline=2
MarginV=60
WrapStyle=0
```

**智能断句规则（保留v5.1）**：
- 单行字幕 **≤ 15 字**（含标点）
- 遇到标点时**自动断句**
- **禁止行尾出现标点**

### #7 视频素材（v5.3 强化：≥2段真实视频）

### #8 封面规格

## 构建后验收代码（必须执行）

```python
def post_build_check(video_path, srt_path):
    """v7.1 完整验收检查（含感知去重）"""
    checks = []
    
    # 0. 感知去重检查（v7.1 新增）
    dupes = run_dedup_v3()
    checks.append(("L1+L2去重无重复", len(dupes) == 0))
    
    # 1. 分辨率横屏
    w, h = get_resolution(video_path)
    checks.append(("分辨率1920x1080", w == 1920 and h == 1080))
    
    # 2. 文件大小
    checks.append(("文件>=1MB", os.path.getsize(video_path) > 1*1024*1024))
    
    # 3. 时长
    duration = get_duration(video_path)
    checks.append(("时长30-70秒", 30 <= duration <= 70))
    
    # 4. 字幕单行检查
    with open(srt_path, 'r', encoding='utf-8') as f:
        content = f.read()
    checks.append(("字幕无\\n换行", '\\n' not in content.replace('\\r\\n', '\\n')))
    
    # 5. 抽帧检查字幕显示
    for t in [3, duration//2, duration-5]:
        frame = extract_frame(video_path, t)
        checks.append((f"帧{t}s字幕正常", has_single_line_subtitle(frame)))
    
    # 6. 视频素材数量
    video_clips = count_video_clips_in_project()
    checks.append(("视频素材>=2段", video_clips >= 2))
    
    return checks
```
