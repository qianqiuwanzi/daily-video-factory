| L2 | 感知指纹（图像数据区采样MD5） | 同图不同压缩/格式 | 内容相似但不同图 | P2完成后, P3合成前 |
| L3 | 视觉抽检（AI/人工目视） | 语义不匹配、内容相似 | 需要理解含义 | P4交付前 |

#### #0-B L2 感知指纹算法（Node.js 实现）

```javascript
// dedup_v3.js 核心逻辑
const crypto = require('crypto');
function perceptualFingerprint(buffer) {
  let sample;
  if (buffer[0] === 0x89) { // PNG: skip 8-byte header + IHDR(13B)
    sample = buffer.slice(25, 2073); // 跳过元数据，取2KB图像数据
  } else if (buffer[0] === 0xff) { // JPEG: skip header to image data area
    sample = buffer.slice(400, 2400); // 跳过EXIF等，取2KB图像数据
  } else {
    sample = buffer.slice(0, 2048);
  }
  return crypto.createHash('md5').update(sample).digest('hex').slice(0, 12);
}
```

### #5 动态效果（v9.0 更新：解决画面晃动 + 强制语义匹配）

#### 🔴 晃动问题根因与修复（v9.0 新增）

**问题**：`z='min(zoom+0.002,1.3)'` 在 30fps 下每帧+0.002，1秒就+6%，2.5秒就触达1.3倍上限被截断，然后 zoompan 重置 zoom 值导致画面"跳一下"。

**解决方案（v9.0）**：将 zoom 变化率降为原来的 **1/10**，即每帧±0.0002，同时延长 zoom 起始缓冲（避免开头突兀）：

| 参数 | v4.0 旧值（晃） | v9.0 新值（稳） |
|------|----------------|----------------|
| zoom-in 增量/帧 | `+0.002` | `+0.0002` |
| zoom-out 减量/帧 | `-0.002` | `-0.0002` |
| zoom-pulse 幅度 | `0.06*sin(ON)` | `0.008*sin(ON)` |
| diagonal 偏移 | `x-0.3*on, y-0.2*on` | `x-0.04*on, y-0.03*on` |
| pan 偏移量 | `0.5*on` | `0.08*on` |

**效果对比（以 2.5s、30fps 图片场景为例）**：
- 旧：2.5s × 30fps × 0.002 = 150步 = 从1.0→1.15（触发上限1.3重置→跳帧）
- 新：2.5s × 30fps × 0.0002 = 15步 = 从1.0→1.003（全程平稳）

### #14 素材尺寸验收（v11.9 新增 🔴🔴🔴 防止竖屏视频混入 P3）

**事故根因（2026-05-12）**：P2b 下载了 scene 9 的所有候选 URL，包括竖屏视频 `BV1EwRQBLEaz.mp4`（9:16），未经验证直接进入 P3。

**强制规则**：
- ✅ **每个视频素材**（.mp4）：ffprobe 验证 `width > height`（横屏）
  - 命令：`ffprobe -v error -select_streams v:0 -show_entries stream=width,height -of csv=p=0 "video.mp4"`
  - 解析：`480,852` → height > width → **竖屏 → 不合格**
- ✅ **每个图片素材**（.jpg/.png）：PIL 验证 `width > height`（横屏）
- ✅ **最低分辨率**：宽×高 ≥ 1280×720
- ❌ **竖屏视频（height > width）**：立即删除，不得进入 P3
- ❌ **尺寸不达标**：删除 → 重新搜索 → 重下 → 最多3次 → 仍失败报告用户

**验收时机**：
- P2b 下载完成后，**进入 P3 之前**，必须执行尺寸验收
- P2b 的第⑧项门禁（第8项，v12.5 新增）

**验收执行（Python 参考）**：
```python
import subprocess, os
from PIL import Image

def check_video_orientation(path):
    result = subprocess.run(
        ['ffprobe', '-v', 'error', '-select_streams', 'v:0',
         '-show_entries', 'stream=width,height',
         '-of', 'csv=p=0', path],
        capture_output=True, text=True
    )
    w, h = map(int, result.stdout.strip().split(','))
    return w > h, w, h  # is_landscape, width, height

def check_image_orientation(path):
    img = Image.open(path)
    w, h = img.size
    return w > h, w, h

for f in os.listdir('assets/video'):
    if f.endswith('.mp4'):
        ok, w, h = check_video_orientation(f)
        if not ok:
            print(f"DEL {f} (竖屏 {w}x{h})")
            os.remove(f)  # 竖屏删除
        elif w * h < 1280 * 720:
            print(f"DEL {f} (太小 {w}x{h})")
            os.remove(f)  # 太小删除
```

def post_build_check(video_path, srt_path):
    """v11.8 完整验收检查（含视频MD5去重 + 感知去重）"""
    checks = []
    
    # 0. 感知去重检查（v7.1 图片去重）
    dupes = run_dedup_v3()
    checks.append(("L1+L2图片去重无重复", len(dupes) == 0))
    
    # 0b. 视频素材MD5去重检查（v11.8新增 — 拦截同一视频复制重命名）
    video_md5_map = {}
    video_dupes = []
    for mp4 in glob.glob(os.path.join(assets_dir, '*.mp4')):
        md5 = hashlib.md5(open(mp4, 'rb').read()).hexdigest()
        if md5 in video_md5_map:
            video_dupes.append((video_md5_map[md5], mp4))
        video_md5_map[md5] = mp4
    checks.append(("视频MD5无重复", len(video_dupes) == 0))
    if video_dupes:
        print(f"❌ 视频MD5重复: {video_dupes}")  # 例如：[('scene_03.mp4','scene_08.mp4'), ...]
    
    # 1. 分辨率
    w, h = get_resolution(video_path)
    checks.append(("分辨率正确（横屏1920x1080或竖屏1920x1920）", (w == 1920 and h == 1080) or (w == 1080 and h == 1920)))
    
    # 2. 文件大小
    checks.append(("文件>=1MB", os.path.getsize(video_path) > 1*1024*1024))
    
    # 3. 时长
    duration = get_duration(video_path)
    checks.append(("时长45-120秒", 45 <= duration <= 120))
    
    # 4. 字幕单行检查
    with open(srt_path, 'r', encoding='utf-8') as f:
        content = f.read()
    checks.append(("字幕无\\n换行", '\\n' not in content.replace('\\r\\n', '\\n')))
    
    # 5. 抽帧检查字幕显示
    for t in [3, duration//2, duration-5]:
        frame = extract_frame(video_path, t)
        checks.append((f"帧{t}s字幕正常", has_single_line_subtitle(frame)))
    
    # 6. 视频素材数量
    video_clips = count_video_clips_in_project()
    checks.append(("视频素材>=2段", video_clips >= 2))
    
    return checks
